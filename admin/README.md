# 充电桩后台统计系统

## 项目简介

这是一个充电桩后台统计系统的前端demo，用于展示充电桩的使用数据统计和分析。系统提供了数据概览、用户分析、充电桩分析、时段分析、地域分析和数据导出等功能。

## 技术栈

- HTML5
- CSS3
- JavaScript
- ECharts 5

## 项目结构

```
admin/
├── css/
│   └── style.css          # 样式文件
├── js/
│   ├── mock-data.js       # 模拟数据
│   ├── index.js           # 主页面脚本
│   └── users.js           # 用户分析页面脚本
├── index.html             # 数据概览页面
├── users.html             # 用户分析页面
└── README.md              # 项目说明
```

## 功能模块

1. **数据概览**：展示总使用次数、总充电量、总用户数、平均充电时长等核心指标，以及24小时使用分布、区域使用热力图、充电桩使用率排名和月度充电量趋势等图表。

2. **用户分析**：展示用户增长趋势、充电时长分布、用户使用频率和用户类型分布等数据。

3. **充电桩分析**：（待实现）展示充电桩使用率、故障分析、充电量分析等数据。

4. **时段分析**：（待实现）展示24小时使用分布、周/月使用趋势、节假日影响等数据。

5. **地域分析**：（待实现）展示区域分布热力图、区域对比、热点区域分析等数据。

6. **数据导出**：（待实现）支持导出Excel、PDF格式的统计报表。

## 如何使用

1. 打开 `admin/index.html` 文件即可访问后台统计系统。

2. 在左侧导航栏可以切换不同的功能模块。

3. 在顶部可以选择不同的时间范围（今日、本周、本月、本年）查看数据。

4. 点击「刷新数据」按钮可以刷新页面数据。

## 数据说明

系统使用的是模拟数据，存储在 `js/mock-data.js` 文件中。实际使用时，需要替换为真实的后端API数据。
